# 🧮 SmartCalc — Advanced Android Calculator with History & Scientific Modes

SmartCalc is a modern, responsive Android application built using Kotlin and XML. It features a seamless transition between a clean, daily-use simple calculator and a powerful scientific layout, complete with an interactive calculation history tracker. Developed as a self-initiated project for the **Mobile Application Development** course.

---

## 🤖 AI-Directed Engineering (Iterative Prompting Workflow)
While generative AI assisted in writing the syntax, this application's core architecture and final execution were entirely driven by my rigorous iterative direction:
* **The Process:** The development started from a very basic boilerplate. I audited the AI's output line-by-line, pushed back against generic code blocks, and micro-managed the feature roadmap.
* **My Architectural Inputs:** I manually directed the evolution from a simple layout to a dual-mode dynamic state, requested the custom upper-screen history logic, enforced layout fixes for visual consistency, and guided all mathematical state constraints. 

*This project showcases my ability to act as a Product Manager and Technical Director—knowing how to architect a solution, debug logic, and iteratively guide AI tools to achieve a complex, production-ready outcome.*

---

## 📱 Key Features

- **Dual-Mode Dynamic Interface:** Easily switch between a standard basic view and a comprehensive scientific dashboard with a single tap.
- **Persistent Calculation History:** Real-time history log displayed at the top, allowing users to track previous computations effortlessly.
- **Standard Mathematical Operations:** Flawless handling of foundational arithmetic: Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
- **Advanced Scientific Suite:** Fully equipped with high-level functions, including Square Root (`√`), Trigonometry (`sin`, `cos`, `tan`), Logarithms, Exponents, and more.
- **Robust Error Handling:** Built-in validation to prevent crashes from invalid mathematical expressions (e.g., division by zero).
- **Responsive XML Layouts:** Pixel-perfect user interface optimized for varying Android screen sizes and orientations.
